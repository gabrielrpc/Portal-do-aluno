# Portal do Aluno

## Descrição

O **Portal do Aluno** é uma aplicação de gerenciamento de alunos desenvolvida em C# utilizando Windows Forms, LINQ e Entity Framework com PostgreSQL. Atualmente, a aplicação permite listar os alunos cadastrados e adicionar novos alunos.

## Funcionalidades

- **Listar Alunos**: Visualize a lista de alunos cadastrados no sistema.
- **Cadastrar Novo Aluno**: Adicione novos alunos ao sistema.

## Tecnologias Utilizadas

- **Linguagem de Programação**: C#
- **Interface Gráfica**: Windows Forms
- **ORM**: Entity Framework
- **Banco de Dados**: PostgreSQL
- **Consultas**: LINQ
